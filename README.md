# ByeSpec

ByeSpec is a VSCode plugin to temporarily dim Specs in Elixir source files.

When active it fades and folds Elixir `@spec` lines and blocks:

Screenshots: (TBD)

# Options

- VSCode commands: force show and hide
- VSCode configuration:
  - opacity
  - backgroundColor
  - plugin on/off
  - fold on/off
  - gutter icon on/off
